# .github — Cyber Elders organization profile

This repository powers the **Cyber Elders** GitHub organization profile and shared
community defaults.

- `profile/README.md` — the org landing page (renders at <https://github.com/Cyber-Elders>
  **once this repo is public** — planned for the first public launch / Wave 1).
- `profile/assets/` — brand images: `banner.png` (Elder Mind banner), `avatar.png`
  (the org avatar / elder mark), `social-card.png` (link-preview, 1200×630).
- `tools/build_org_card.py` — regenerates the org social card (pure-Python / Pillow).

Brand: deep indigo + the green→blue→orange→purple maturity gradient + the elder mark.
